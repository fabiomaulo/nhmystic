To use the project you have to create a folder named "Lib" at same level of NHibernateMystic.sln

There you have to copy files deployed with ConfOrm and files deployed with NHibernate

ConfOrm downloads : http://code.google.com/p/codeconform/downloads/list

NHibernate downloads : http://sourceforge.net/projects/nhibernate/files/
